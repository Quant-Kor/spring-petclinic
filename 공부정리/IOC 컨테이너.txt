ApplicationContext가
BeanFactory를 상속받고 있음

IOC 컨테이너가 하는 일
빈(bean)을 만들고 엮어주며 제공해준다.

클래스 옆의 녹색 표시가 되어있으면 빈으로 등록이 되어있다는 것이고 녹색표시가 없으면 빈으로 등록 X

의존성 주입은 Bean 끼리만 가능함(IOC 컨테이너 안에 것들만 의존성 주입을 해줌)

실습내용 중 Applicationcontext를 직접 사용할 일을 없고 Application context가 담고 있는 bean들을 알아서 주입을 해줌(getbean을 해서 주입을 할 필요가 없음)
getbean을 사용해 가져온 application context의 ownerrepository랑 인젝션을 통해 받은 ownerrepository랑 해시가 같음
public String bean() {
		return "bean: " + applicationContext.getBean(OwnerRepository.class) + "\n"
			+"owners: " + this.owners;
	}
